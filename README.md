# Patrika Media & Marketing Enterprise

Premium enterprise website built with Next.js 16, React 19, TypeScript, Tailwind CSS, and an MDX-backed content system.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Included

- Homepage and core marketing pages
- Dynamic services, industries, case studies, and blog routes
- MDX-backed blog collection with 20 launch articles
- Contact form server action with validation and optional Resend notifications
- Sitemap, robots, metadata, and structured data support
